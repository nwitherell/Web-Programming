The way I would define it is to have a CSS for the average case in which the website would be setup. I would set class tags for bg color, department style, and the way tables are set to look like.
The CSS would be easy to modify, as each tag would have universalr properties, but be easily modifieble for each departments needs .background - background color
.departmentStyle - each departments style
.configureTable - used for non-specified tables
.exam - for online tests
.calender - for the overall schedule
.sheet - for average online documents to show off
.timeTable -for schedule on a per day basis
.header - the header
.title - for the name of the class or department
.info - for general info in each class
